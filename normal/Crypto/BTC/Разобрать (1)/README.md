# bitcoin-fork-checker
Bitcoin forks balance checker 
![Image](https://i.imgur.com/WKA0jgX.png)

### support
Bitcoin Cash, Bitcore(BTX), SuperBTC, B2X, Lightningbitcoin, BCX, BTG

### how to
```bash
python main.py -h
```
```bash
python main.py filepath -t <number threads>
```
